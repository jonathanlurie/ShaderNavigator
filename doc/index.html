ChunkCollection

The Chunk Collection is the container for all the chunks at a given resolution level. The resolutionLevel goes from 0 to 6, 0 being the poorer and 6 the sharper. ChunkCollection should not be asked anything directly, LevelManager should be the interface for that.

▸ get8ClosestTextureData(position)

return the 8 texture data for the 8 closest chunks to the given position. The texture data are ordered so that if some are invalid (out of bound, fake texture) they will be placed in the end of the array.

get8ClosestTextureData(position: Array): object
Parameters
position (Array) An arbitrary position in world coordinates.
Returns
object: the object contains 1 array of textures, one array of chunks origins and one integer to inform how many of these textures are valid. This may seem a strange data structure but it's the closest from what can take the fragment shader and will require no further work. Remember we are potentially speaking of thousands of chunks being loaded!
